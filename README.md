# [WIP] discobolus
## FOSS Discord bot with full and total customisability for free.
Made with love in 
- `Typescript`
- `NodeJS`
- `Eris`
- `Express`
- `React`
- `MySQL (MariaDB)`

How to run:
- Clone the repository: ```git clone http://github.com/vecondite/discobolus```
- Install tsx (to run Typescript): ```pnpm install -g tsx```
- Install packages: ```pnpm install```
- Run the Discord Bot: ```pnpm exec tsx apps/bot/src/index.ts```

Notes:
- Frontend, Backend and API all have to be run seperately as of now.
- Only the backend is somewhat functional at the moment. Neither the API nor the Frontend are not developed yet.
- Please do send a pull request or open an issue if you find any bugs or vulnerabilities.

Thank you.
